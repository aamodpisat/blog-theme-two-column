# Setup
* Download the theme
* Extract the downloaded theme and place 'two-column' folder inside your themes folder of your application.
* Activate this theme in config/all.js as shown below

    ```
     {
        theme : 'two-column'
     }
    ```
* Run the Application

    ``` npm start ```
